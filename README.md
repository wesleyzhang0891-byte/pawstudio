**欢迎使用 Pawnote**

它是一款常驻菜单栏的极简速记 + 番茄闹钟 + 白噪音的工具，极致轻便

**亮点**
1、很克制：只有6个速记页，极致的简约（支持Markdown）
2、番茄钟：25分钟一次专注，干活摸鱼都在行
3、白噪音：几十首白噪音，带薪发呆也是极好的
4、打字机：右键切换到全屏打字，牛马包爱的
5、自隐藏：没事儿自动消失，有事儿随时唤起
6、易切换：触控板轻划双指，丝滑切换不同日记
7、多皮肤： 六套深浅色皮肤，总有一套你喜欢
8、自存储：毫秒级存储、本地备份，数据丢了你找我

**基础**
- 顶部 6 个彩色点 = 6 个独立速记页
- 切换页面：点圆点 / ⌘1–⌘6 / ⌥← ⌥→ / 触控板双指左右滑
- 点击菜单栏猫爪图标或按 ⌘E 随时呼出 / 收起
- 编辑时按 Esc 可隐藏面板，下次呼出回到默认位置
- 左上角图钉可置顶（不随点击外部消失）
- 内容自动保存，崩溃也能恢复，文件以 Markdown 存储

**格式（所见即所得）**
- 加粗 ⌘B、斜体 ⌘I
- 行首输入 `# ` 变标题；⌃⌘0 改回正文，⌃⌘1/2/3 设标题
- 复制 / 粘贴 / 剪切 / 撤销：⌘C / ⌘V / ⌘X / ⌘Z（右键菜单为中文）
- 从外部粘贴 Markdown 会自动识别格式
- 字号：⌘+ 放大、⌘- 缩小（全局统一，文本自动重排）

**专注**
- 全屏沉浸写作：⌃⌘F 或 右键「全屏模式」铺满整屏，沉浸式打字，Esc 退出还原
- 番茄钟：点时间开始，再点暂停（暂停时闪烁），⏹ 停止；到点提醒休息
- 白噪音：数十首音乐，点右上角音乐图标随机播放；支持 Mac 媒体键（播放/上一首/下一首）控制

**外观**
- 6 套配色主题：暖纸 / 晨雾 / 午夜黑 / 暗琥珀等，在设置里切换
- 面板可拖动、可缩放并记忆尺寸；圆角设计

**存储与安全**
- 自动保存：停止输入 0.5 秒即存，连续输入最长 5 秒强制存一次，无需手动保存
- 原子写入：先写临时文件再原子替换，断电/崩溃也不会出现损坏的半个文件
- 崩溃恢复：未及时保存的内容会暂存，重开时提示恢复
- 完整性指纹：每次保存记录 SHA-256 校验码，可用于核对内容是否被外部改动
- 自动备份：定期为每个点生成历史快照，可回溯
- 不锁定：正文以纯 Markdown 文件存储，可用任何编辑器打开
